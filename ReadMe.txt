TheEye
--------------------
	A software utility to identify anomalous behaviour from a camera feed. This is an attempt to use machine learning combined with image processing in order to make camera intelligent so that they can notify on detecting an anomalous behaviour. An anomalous behaviour, herein, is defined as an activity that seldom occurs in the camera's view for example, an accident on a highway. It uses WebCamFeed Extractor tool as a data acquisition software that extracts video feed from one of the public cameras at Iowa State University and saves it as a video. 
	
WebCamFeed Extractor
----------------------
	This tool extracts video feed from one of the public cameras at Iowa State University and is the data acquisition software for the project.
